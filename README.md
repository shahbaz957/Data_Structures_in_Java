# Data Structures in Java

A curated Java repository focused on learning and practicing core **data structures** through clear, beginner-friendly implementations and examples. 
The goal is to provide a practical reference for common structures used in interviews, competitive programming, and everyday backend/software development. 

---

## Contents

- Arrays
- Linked List (Singly / Doubly) 
- Stack 
- Queue 
- HashMap / HashSet 
- Trees (BST / AVL / Heap) 
- Graphs 
- Tries

---

## Tech Stack

- Language: Java 
- IDE: IntelliJ IDEA /  VS Code 

---

## Repository Goals

- Strengthen fundamentals of data organization and access patterns. 
- Provide clean implementations that are easy to read and modify.
- Serve as a revision resource for interviews and exams.
